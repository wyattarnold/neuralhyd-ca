Data
====

Training Data
-------------

The dataset covers **216 basins** across 3 hydroclimatic tiers — Tier 1
(88 warm, rainfall-dominated), Tier 2 (97 transitional rain-snow), and
Tier 3 (31 cold, snow-dominated).

- **Climate forcing**: Daily precipitation (mm), tmax (°C), tmin (°C),
  area-weighted from gridded sources. Records span 1915–2018 (~38k days).
- **Streamflow**: Daily records from USGS, variable per basin (typically
  1950s–present). Converted from cfs to mm/day using basin area.
- **Static attributes**: 16 features derived from BasinATLAS and climate
  statistics — watershed geometry (area, slope), land cover (forest
  fraction), soil texture, river network characteristics,
  geology/lithology classes, and long-term climate normals (mean
  precipitation, PET, aridity index, snow fraction).

Normalisation
-------------

- **Flow targets**: Converted cfs → mm/day using basin area, then divided
  by per-basin std. This removes area as a confound and makes flow
  physically comparable across basins.
- **Climate inputs**: z-score normalised globally using training-basin
  statistics only.
- **Static attributes**: z-score normalised globally using training basins.
  Selected features with heavy right skew (e.g. area) are log-transformed
  before normalisation.

Validation
----------

5-fold stratified spatial cross-validation. Basins — not timesteps — are
the unit of splitting. Each fold holds out ~20% of flow-days per tier.
**No watershed appears in both train and validation within a fold.** Every
basin is evaluated exactly once as if ungauged, testing the model's ability
to generalise to basins it has never seen.

Primary metrics are per-tier median NSE, KGE, FHV (peak flow bias), and
FLV (low flow bias).

External Comparison
-------------------

Process-based model outputs from the California Energy Commission (CEC)
are available for overlapping basins, enabling direct comparison between
data-driven and physics-based approaches.

The comparison data comes from the **VIC (Variable Infiltration Capacity)** model, 
generated by UCLA as part of the CEC's Climate Data and Analysis Working Group (`C-DAWG
<https://www.energy.ca.gov/programs-and-topics/topics/research-and-development/climate-data-and-analysis-working-group-c-dawg>`_).
These models were forced with a subset of LOCA2-hybrid and dynamically
downscaled WRF climate data to produce daily hydrologic simulations across
California.
